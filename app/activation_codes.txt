# يحتوي هذا الملف على قائمة بأكواد التفعيل غير المستخدمة.
# يجب أن يكون كل سطر بالتنسيق التالي: CODE,plan_type
# مثال: MTH-12345ABCDE,monthly

# ------------------------------------------------------------------------------
#
# 1. لإضافة أكواد جديدة:
#    python manage_codes.py add --plan <plan_type> --quantity <number>
#
#    <plan_type> يمكن أن يكون: monthly, yearly, lifetime
#    <number> هو عدد الأكواد التي تريد إنشاءها.
#
#    مثال:
#    python manage_codes.py add --plan monthly --quantity 50
#
# 2. لعرض جميع الأكواد غير المستخدمة في هذا الملف:
#    python manage_codes.py list
#
# 3. لحذف كود معين:
#    python manage_codes.py delete <code>
#
#    <code> هو كود التفعيل الكامل (مثال: MTH-12345ABCDE).
#
#    مثال:
#    python manage_codes.py delete MTH-12345ABCDE
#
# 4. لرفع جميع الأكواد الموجودة في هذا الملف إلى قاعدة البيانات:
#    python import_codes.py
